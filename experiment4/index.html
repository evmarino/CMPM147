Experiment 4 - Three Infinite World Generators

Description

This experiment showcases three different procedural generators: a Movie Recommendation Infinite Picker, an Ant Killing Generator, and a Dot Cleaning game. Each generator responds dynamically to clicks and updates the world in fun and creative ways.

Technical

Using p5.js, I implemented a world key system based on XXHash hashing and seeded noise/random generators. The movie cube fetches random movie data from TMDB API when clicked, the ant generator simulates procedural movement, and the dot generator requires cleaning randomly scattered dots. All systems share a world coordinate space and dynamic drawing based on seed states.

Reflection

I really enjoyed implementing different kinds of generators in the same visual framework. Managing multiple procedural systems helped me better understand world seeding, modularity, and dynamic drawing techniques. I also learned the importance of thinking about user interaction styles like click vs hold across different generators.
