Description

In this assignment, we build a generator that evolves abstractions toward the three inspiring photographs. You can switch images, control the mutation rate, and watch two canvases side by side: the live mutation (Active) and the best so far (Best). Under is the memory strip collecting past winning snapshots.

Technical

  • Load and display three images, one being the initial reference.
  • Initialize an abstract design for each image.
  • Render, score, and anneal via Gaussian mutations.
  • Dropdown menu for images, restart button, mutation rate slider, and score display.

Annealing

Start with the slider at 100% and hit Restart to see different abstractions. Slowly drag the slider down toward 0%, and see how the designs lock in as the mutation steps become ever smaller.
